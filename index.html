PiggiesCSS

A nice, class-less, feature-packed CSS library that offers its own CSS reset/normalizer, and styles for accessibility and themes (via :root variables). It is meant to be built off of (like all other CSS libraries.

This page right now is using PiggiesCSS! To use it in your own site, put this code in it:

<link rel="stylesheet" href="https://jeffreymaniac.github.io/PiggiesCSS/piggies.css" />
